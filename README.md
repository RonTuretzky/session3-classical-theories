# Session 3 — Theories of Communalism I

`session-03-classical-theories.html` is a self-contained Reveal.js deck. No build step.

## Present
Open the file in Chrome/Safari. Then:
- `f` — fullscreen
- `s` — speaker view (notes + next-slide preview, on a second screen ideally)
- `←` / `→` / space — navigate
- `o` or Esc — slide overview
- `b` — black out the screen

## Export to PDF
Open the deck with `?print-pdf` appended:
`file:///.../session-03-classical-theories.html?print-pdf`
Then ⌘P → Save as PDF, "Background graphics" on, margins None.

## Timing target
~75 slides over ~150 minutes of teaching, including two 5-minute breaks and a 12-minute writing exercise. Block-by-block timing is in the speaker notes.

## Edit
Each `<section>` is a slide. Speaker notes live in `<aside class="notes">`. Quotes use `<blockquote>` with a `<cite>`. Add slides by copying an existing `<section>`.

## Per-slide comments
A textarea pinned to the bottom of the screen lets you write a comment on each slide.
- Auto-saved to your browser's localStorage as you type
- Press `c` (or click the bar) to hide/show the panel
- The final "Thank you" slide has a **Copy all comments** button — copies a Markdown dump of every commented slide (with titles) to your clipboard
- Comments persist across reloads. To wipe them, run `localStorage.removeItem('session3-comments-v1')` in the browser console.
- The panel is hidden in PDF export.
